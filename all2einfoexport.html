Collected Information Export UI

Last updated: Friday 12 June 2009 13:02

UNIX name Owner Status Version Compatible with
all2einfoexport Norman Leutner stable 1.0.1 4.x
Adds a tab to the administration interface wich enables a functionality to export collected information as xml files

The collected information can be exported for each information collection class, data range, since last export or full export. (See Screenshot for more details)
For example the exports can be used with Microsoft Excel (since Version 2003) as XML Table for further usage.

Hint
Selected options are parsed with the values.
