BC TopLevelNode

Last updated: Friday 16 December 2011 10:12

UNIX name Owner Status Version Compatible with
bctoplevelnode Brookins Consulting stable 0.0.3 4.x -> 4.6+
Command line scripts to create node in top level root node or move existing node to top level root node.

Primary features

The example bctoplevelnodecreate.php and bctoplevelnodemove.php command line scripts provided

Summary

These scripts can create a top level node or move an existing node to the top level root node

Distribution

This source code is distributed publicly as a solution by Brookins Consulting on github.com

https://github.com/brookinsconsulting/bctoplevelnode

Please feel free to use this solution freely.

Usage

See doc/README for complete eZ Publish PHP command line script example usages provided by extension. Here is the primary usage to create a top level node:

./extension/bctoplevelnode/bin/php/bctoplevelnodecreate.php --siteacess ezwebin_site_user --admin-user=admin --class-id=1 --name=ExampleNodeName001
 

Here is the primary usage to move an existing node to the top level node:

./extension/bctoplevelnode/bin/php/bctoplevelnodemove.php --siteacess ezwebin_site_user --admin-user=admin --node-id=4400

How to install extension

- Download the code for this extension from github.com into the extension directory,
https://github.com/brookinsconsulting/bctoplevelnode

There is no need to configure ini settings or regenerate autoloads or even clear caches, these command line scripts run right out of the box without configuration!

Compatibility

This solution is compatible with eZ Publish 4.x -> eZ Publish Community Build 2011.10 and PHP 5.x
