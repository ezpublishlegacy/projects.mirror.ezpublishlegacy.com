Hi Xavier,

since we are using sitemaps in our projects, the traffic and from the google bots have increased and are indexing the sites more frequently.

Project mobello.de:
April 2008 (without sitemap):
1000-4000 daily index sites

Juli (with cronjob based sitemap)
Max daily: 8122
average daily: 5517

The index sites have also increased from 6.000 sites to more than 10.000 sites.
So we have much more traffic on that project.

The code itself within the extension is very simple:
- fetch a defined tree
- generate xml dom
- save to file

Due to setting priorities for all pages may become very complex,
I made the experience it's better so let google handle priorities of the pages.
Usually this is done by the number of visits on that specific page.

The priority itself is not implemented within the extension yet.
We may discuss about that and implement it if someone needs it.
