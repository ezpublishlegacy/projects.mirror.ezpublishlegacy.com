AdminAid

Last updated: Saturday 26 November 2011 19:36

UNIX name Owner Status Version Compatible with
adminaid Arne Bakkeb√∏ stable 1.0 4.x
Functionality to log in as any user from admin user, without password. Also gives a detailed overview of content object data, as well as a list of content objects based on specified class.

This extensions source code has been moved to Github. Feel free to download the latest source from there: https://github.com/arnebratt/AdminAid

AdminAid started as a simple module to be able to switch users
easily while debugging. After that I decided to extend it a little,
and make a proper extension out of it. Today you can also view
detailed content object data, and a list of all objects based on
a class. More functionality may come in the future.

Please read the doc/INSTALL.txt for details.
