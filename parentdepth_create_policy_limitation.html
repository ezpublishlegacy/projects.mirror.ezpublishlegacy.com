ParentDepth create policy limitation

UNIX name Owner Status Version Compatible with

This patch will add a ParentDepth policy limitation to content/create.

This patch will add a ParentDepth policy limitation for content/create.
Just like ParentClass limits object creation to parent nodes of the
selected classes, the ParentDepth limitation limits object creation
to parent nodes with the selected depths.
