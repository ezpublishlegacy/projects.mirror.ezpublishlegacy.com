eZ Flash Tag Cloud

Last updated: Monday 12 September 2011 12:57

UNIX name Owner Status Version Compatible with
ezflashtagcloud Nicolas Lescure stable 1.1 4.0+
Display a sexy tag cloud in flash using the keywords of eZ Publish

You can display a sphere of tags based on the eZ Publish keywords.
The sphere rotates according to your mouse. When you click on a tag, you go to the page that lists all related contents, just as the ezwebin tag cloud does.

For SEO, all links are also in the html code.

This extension is based on the ezwebin template operator to fetch tags and on the the WP-Cumulus : http://plugins.svn.wordpress.org/wp-cumulus/tags/1.23/

You don't need ezwebin to use it.
