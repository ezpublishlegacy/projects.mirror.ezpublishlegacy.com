eZchat

Last updated: Monday 24 March 2008 18:48

UNIX name Owner Status Version Compatible with
ezchat Xavier Gouley alpha 0.1 4.0.0
An implementation of blueimp ajax chat in eZpublish, like an extension with module and datatype views, and complete integration of user/role management within eZpublish.

This is the implementation of Ajax Chat 0.8.1.2, by blueimp (https://blueimp.net/ajax/), in an extension.

This chat in ajax allow multiple channel, 3 role levels, and typical functionalities for a chat server.

This implementation allow us to create new channel with content (with a ezchatroom datatype), to administrate roles with eZpublish, and to manage chat users with eZpublish users.

Functionnalities with personal image, Google Maps location were added. Video streaming or else will be planned soon.
