Hi Hachem,

Try to change this code in "kernel/content/collectinformation.php":



if (file_exists( eZSys::wwwDir()."/".$Filename )) {


       $fileAttachments[]=array(eZSys::wwwDir()."/".$Filename, $OriginalFilename, $FileType);


}


for this:



if (file_exists( $Filename )) {


       $fileAttachments[]=array($Filename, $OriginalFilename, $FileType);


}
