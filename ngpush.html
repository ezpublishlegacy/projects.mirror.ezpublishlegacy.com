Netgen Push

Last updated: Friday 22 November 2013 14:36

UNIX name Owner Status Version Compatible with
ngpush Petar Å panja stable 1.3 4.3+
Netgen Push provides a simple and effective way to push any kind of content from eZ Publish to various social networks.

For the time being, Twitter and Facebook connectors are implemented, while eZ Publish REST API and Tumblr API are in the planning stage. Netgen Push also provides a basic framework to develop your own connectors to any social network that uses OAuth authentication protocol.

You can find detailed installation and usage instructions on our blog: http://www.netgen.hr/eng/Blog/Overview-of-Netgen-Push-extension

Source code is available at GitHub.
