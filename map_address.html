Map Address module
===========================
This module introduces a custom tag "map_address" that uses the google
map API
to map an address.
Requirements
-----------------
eZ publish 3.6.0 and above
Google Maps API Key - signup here
Installation
-----------------
Follow these steps to add the Map Address module to your eZ publish
installation:
1) Extract the archive into the /extension directory
2) Edit site.ini.append in /settings/override. Add the following to the
file:
[ExtensionSettings]
ActiveExtensions[]=map_address
If you already have the [ExtensionSettings] block, just add the second
line.
3) Clear the cache
4) Obtain a Google Maps API Key for your site from
5) Edit extensions/map_address/settings/map_address.ini and
replace
"Enter google map API key here" with your sites key
[Google]
Key=ABQIAAAAeGhVkhTsMrhYLPqrWz...
Usage
-----------------
1) Edit any content onject that contains an XML block.
2) Click on the custom tag button in the Online Editor
toolbar
3) Select "map_address" from the Custom Tag selector
4) Enter the full address into the text field
e.g. "365 Little Collins Street Melbourne Victoria Australia"
and click on OK
5) Finish edits and publish object
Map size and zoom levels can be changed in map_address.ini file.
The extension uses the Google Maps API to map the address. If you
get
javascript errors like "Address not found" then you have to look at the
address
you have entered. Is it a full address including country?
