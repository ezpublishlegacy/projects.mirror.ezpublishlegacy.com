Real-Time Annotation of PDF, Office & Image Files on Your Website

GroupDocs.Annotation for .NET is a downloadable .NET library that provides functionality for displaying, securely sharing and collaboratively annotating 50+ common image and document types. With this add-on, you can easily integrate GroupDocs.Annotation for .NET into your ezPublish website. Once installed, you will be able to embed a universal document viewer widget along with the annotation toolbar on any page within your website.

Key Features & Benefits

1. Annotate any document type, from any web-enabled device. 

GroupDocs.Annotation for .NET allows you and your users to view and annotate literally all common document and image types without the need to install the original software that they were created in. These include, but not limited to: PDF and Microsoft Word documents, Excel spreadsheets, PowerPoint presentations, Visio diagrams, RTF, CSV, TXT and OpenDocument formats, AutoCAD drawings and raster images.

To view and annotate a document embedded to your ezPublish website via GroupDocs.Annotation, all your users will need is a regular web-browser like IE8+, FireFox, Chrome, Opera, Sfari5+ or their mobile versions. There is no need to install any office software or browser plugins.

2. Real-time annotation mode. Several users can annotate the same document at a time. Users can easily invite others for document review, see each otherâ€™s comments and reply to them.

3. Native Word & PDF annotations. Comments, notes and markup placed on Word or PDF documents via GroupDocs.Annotation can be exported to Word or PDF files respectively for further offline review and editing.

4. A comprehensive set of annotation tools. UUsers can add sticky notes, mark up selected text with underlines/strikeouts, add comments to the selected text or any area within the document, drop arrows, draw freehand lines and shapes, redact text and images, etc.

5. Share your documents securely. GroupDocs.Annotation for .NET is designed as a client-server middleware. It doesnâ€™t download documents that you publish on your website to clientsâ€™ machines, but converts them into web-compatible content and then renders it in usersâ€™ web-browsers just like a web-page. And thanks to the Digital Rights Management (DRM) feature, you can enable a read-only mode, meaning users canâ€™t download, print or copy the documents you share in any way.

For an extra layer of security, GroupDocs.Annotation allows you to redact text or certain areas within documents, as well as add watermarks to prevent your content from screen-grabbing.

Installation

Before you proceed, please note:

Unlike the cloud-based version, this add-on integrates the downloadable .NET library. It allows you to store and host documents on-premises, but requires the library to be installed on your server first. A free evaluation copy of the GroupDocs.Annotation for .NET library can be downloaded from the GroupDocs website.
