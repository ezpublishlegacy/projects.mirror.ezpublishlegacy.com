Last updated: Monday 03 December 2012 12:17

UNIX name Owner Status Version Compatible with
ezadmin Björn Dieding@xrow.de stable 1.x 3.5 and higher
This extenstion delivers tools for the admin.

- Backup
- PHPInfo
- User Change ( like 'su' under linux )
- Database webinterface for submitting queries
- Accelerator interfaces ( APC, eAccelerator )
