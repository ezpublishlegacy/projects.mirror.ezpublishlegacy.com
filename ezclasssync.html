ezclasssync

Last updated: Friday 07 June 2013 13:45

UNIX name Owner Status Version Compatible with
ezclasssync Bartek Modzelewski beta 1.0 4.x
Extension allows easier content class sync between different developers of the same project.

It's a tool for developers team - extension allows easy and fast content class sync between different developers of the same project. With a few clicks it's possible to pass content class modifications to other developers.

Example:

One developer is extending content class with new attributes. As other developers working with the same site, but having separate databases, it can cause problem, if code using new attributes is updated from git/svn. Using ezclasssync we can push changes in a class together with extension. By a few clicks, developer can check changes in a class and update his working copy (or not), without loosing any data.
