all2e 1PixelOut Mp3 Flashplayer

Last updated: Wednesday 17 December 2008 15:40

UNIX name Owner Status Version Compatible with
all2e1pixeloutmp3flashplayer Markus Bader beta 0.1 4.0
This extension provides a template operator that displays mp3 files with the 1pixelout flash player. It is basically the ez version of this well-known wordpress plugin.

The 1pixelout mp3 flash player is a flash player for playing mp3 audio files. It is originally a wordpress plugin and therefore well-known.

The special about this player is the grouping of files. If you start playing one file, all others will stop playing. You can also change the skin in any possible way, i.e. the color of each and every element.

The template operator is used as shown:

{mp3player($url, $count)}

$url is the url to the mp3 file (not to an object)
$count is optional and just a consecutive number that is necessary for the grouping

Further information can be gathered from 'installation.txt' in the folder 'docs'.
