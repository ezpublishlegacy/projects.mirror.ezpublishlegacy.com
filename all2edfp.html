Google DoubleClick for Publishers (dfp)

Last updated: Friday 27 May 2011 12:31

UNIX name Owner Status Version Compatible with
all2edfp Norman Leutner stable 1.0 4.x
eZ Publish Google DoubleClick for Publishers (dfp) AdServer implementation

all2edfp implements the Google DoubleClick for Publishers (dfp) AdServer functionality within eZ Publish

Features:

  • easy to setup and install
  • uses template operators and persistent variables to generate ad slots
  • support for ez flow ad blocks (single as well as repeated blocks)
  • ads are loaded at the end of the page and rendered on the page for better user experince and loading times
  • flexible ad slot support for each page (number, size, ...)
  • custom targeting filters (urls, nodes, path, logged_in, ...)
  • url exclude patterns

Requires:

  • ezjscore

For more information about the dfp programm visit:
http://www.google.com/dfp/info/sb/

Support
If you have any questions or if you need support installing the extension or setting up all2edfp with eZ Publish feel free to contact us at: http://www.all2e.com
