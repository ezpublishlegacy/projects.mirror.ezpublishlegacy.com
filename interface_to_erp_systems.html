Interface to ERP-systems

Last updated: Monday 03 May 2004 17:06

UNIX name Owner Status Version Compatible with
interface_to_erp-systems Frank Dege planning N/A N/A

eZ publish is a sophisticated platform for e-business solutions.
The implementation of a fully integrated solution requires an
interface to ERP-systems like Navision, SAP or others.
This extension (silver.erp) offers easy to use functions for integrating
an ERP-system. It uses a SOAP-interface, so ERP-client and eZ publish
can be installed on different systems.

silver.solutions has implemented an universal ERP-connector. This connector transfers
orders from eZ publish to an ERP-system and gets stock-information
back from the ERP. The interface uses an appropriate ERP-adapter for
each ERP-system.
The contribution contains a simple demo ERP-adapter. This adapter shows
how to store an order in a XML-file and returns stockinformation for
a given articlenumber.
The archive contains a PDF-document with a "small guide" for
installation and configuration.
we already implemented an ERP-adapter for Microsoft Business Solutions
Navision, a well-known ERP-system. The ERP-adapter for Navision will
cost money because this extensions is based on a closed source
product.
