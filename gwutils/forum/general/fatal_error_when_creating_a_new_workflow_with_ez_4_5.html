Hello,

I use eZ4.5 Entreprise.

When I try to create a new workflow I get this following error :



PHP Fatal error: Call to undefined function ezi18n() in /ez_publish/extension/gwutils/eventtypes/event/gwupdatehidden/gwupdatehiddentype.php on line 64, referer: http://admin.domain.com/workflow/workflowlist/2


So in /extension/gwutils/eventtypes/event/gwupdatehidden/gwupdatehiddentype.php at line 64 I replaced
ezi18n

by
ezpI18n::tr

and it works !
