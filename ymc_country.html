ymc Country

Last updated: Tuesday 17 January 2006 16:44

UNIX name Owner Status Version Compatible with
ymc_country Daniel Beyer planning 1.1 3.6.4 (3.6.x)

This dataype provides a country drop-down list in your objects.
It is based on the ISO 3166-3 standard, has support for ISO 3166-2, is
fully translatable and build as an extension.

This datatype is not a clone of one of the other existing
"Country"-dataypes, which can be found here and therefore it isn't
copmpatible with one of them.
It stands out from the crowd by using ISO 3166-3 to store its values in
the database and makeing use of the nice translation engine of eZ
publish, being the first country datatype allowing you to provide a
country selection in the language of your site.
It is build as a PnP extension which can be enabled via the admin
interface. There is no need to do any configuration to get the
datatype to work. Refer to the file INSTALL if you have trouble with
installing it.
This datatype has no support for the information collector system in eZ
publish. As we don't make usage of the information collector we won't
add support for it in future releases, but fell free to add it into
the public version on pubsvn.ez.no yourself.
Current available languages are:
-English (GB)
-German
