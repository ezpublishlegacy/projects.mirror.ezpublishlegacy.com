Varnish Control Center

Last updated: Friday 31 July 2009 18:23

UNIX name Owner Status Version Compatible with
all2evcc Norman Leutner stable 1.0 4+
This extension contains a varnish control center for the admin interface.

It also contains a workflow which purges the cache within varnish after publish.

Integrated functions are:

  • Varnish Stats
  • Varnish Status
  • Purging the Varnish caches
  • Start / Stop services
  • Terminal
  • Workflow for automated purging
  • Multiple Domains Support added

Support
If you have any questions or if you need support installing the extension or setting up varnish with eZ Publish feel free to contact us at: http://www.all2e.com
