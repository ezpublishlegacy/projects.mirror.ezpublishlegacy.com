admin2++

Last updated: Sunday 19 December 2010 19:37

UNIX name Owner Status Version Compatible with
admin2pp Damien Pobel stable 0.2 4.4
admin2++ aims to add some features and improvements to the admin2 design used in the admin interface of eZ Publish.

What it does ?

See this screencast on Youtube and/or related posts on my blog.

How to install ?

  1. Download admin2++
  2. Extract the archive in the root of eZ Publish
  3. Enable the extension and make sure ezjscore is enabled too
  4. Optionally copy/paste or include the Apache2 configuration in the doc folder in the configuration if the site
  5. And that's all !
