eZ Migration Tools

Last updated: Sunday 20 June 2010 00:23

UNIX name Owner Status Version Compatible with
ezmigrationtools Jean-Luc Chassaing stable 2.0 4.0 and higher
This extension gives client scripts to performe automated site migration. It can performe the sql and php scripts on all the databases found.

This extension was built to upgrade a multiwebsite installation having a specific database for each website.

In this website database settings were specified in the siteaccess settings.

It runs task wich are :

  • List found dbs
  • Perform db backup (can be disabled in settings)
  • Migrate dbs (applying the sql scripts on each database)
  • apply php scripts (all the php scripts are run on each siteaccess having a specific database)

On specific situation a new task can be very simply added.

Last release 2.0 added many changes

Options have been added when calling the script.

It is now possible to skip a task or abort during the execution.
