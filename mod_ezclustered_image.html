mod_ezclustered_image Jérôme Renard beta 1.0-beta > 3.10
Apache module which fetches image in eZ Publish clusters so index_image*.php should not be needed anymore.

2008/27/11 Update : the module dev is done, the beta-1 release is available in the "Downloads" section

Compatible and tested with MySQL > 4.1 and Apache > 2.2

Should be compatible with PostgreSQL and Oracle as well but not tested though.

Please note that this module is still under heavy development I hope to get everything working fine as soon as a I get some spare time.
