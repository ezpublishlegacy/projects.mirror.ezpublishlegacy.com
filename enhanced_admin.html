Enhanced Admin

Last updated: Wednesday 18 February 2009 14:46

UNIX name Owner Status Version Compatible with
enhanced_admin Frédéric DAVID alpha 1.0 4.x
This extension add some features in eZ publish administration.

Feature

  • Enhance Children Navigation
  • Enhance Class Attribute Priority

Installation

  1. Download the archive
  2. Unpack it in the extension directory of eZ Publish
  3. Activate the extension with the Administration interface in System > Extensions
  4. Regenerate the extension autoloads
  5. Clear the caches
