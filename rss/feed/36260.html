http://projects.ez.no/swark/forum/general/mirror_of_the_swark_extension_on_github Hello eZ Community using swark extension! Brookins Consulting is happy to announce we have created a complete mirror import of the project's extension download sources (for eZ Publish 3.x, 4.x and 5.x) into a git based repository on GitHub. https://github.com/brookinsconsulting/swark We will strive to keep this friendly fork / mirror up to date with any changes that come in the future from the forums and other forks in an effort to remain compatible with the latest versions of eZ Publish. Anyone is free to use (clone) this repository, just watch it for changes, or fork this new repository and use it as needed freely. This is a friendly mirror of the extension. Currently the latest version provided by the mirror is from Apr 2, 2012 and will work with the latest version of eZ Publish (Legacy Stack).
