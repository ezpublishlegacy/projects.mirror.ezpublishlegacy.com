This is the first version compatible with eZ Publish 4 and PHP5. I would like to thank <a href="http://projects.ez.no/users/community/kaare_koehler_hoevik__1" target="_self">Kåre Køhler Høvik</a> who initiated the portage to PHP5. This version also brings small bugfixes and a better debug output to the cronjob script.</p> <p>eZVideoFLV is an extension providing a datatype to store, handle and convert to FLV video files. It's based on <a href="http://ez.no/doc/ez_publish/technical_manual/3_9/reference/datatypes/media" target="_self">the default ezmedia datatype</a>. It mainly adds three features : </p> <ul> <li> generate an image preview for each video file using <a href="http://ffmpeg-php.sourceforge.net/" target="_self">ffmpeg-php</a> and <a href="http://ffmpeg-php.sourceforge.net/apidoc.php#ffmpeg_frame" target="_self">GD</a> </li> <li> detect and store video size width ffmpeg-php too </li> <li> convert video file to FLV on upload or with a cronjob script in order to be readable with a flash FLV player (like on Youtube or Dailymotion) using <a href="http://ffmpeg.mplayerhq.hu/" target="_self">ffmpeg</a> program </li> </ul> <p>The extension is bundled with <a href="http://flv-player.net/" target="_self">FLV Player</a> which is released under the MPL 1.1 license. The rest of the extension is released under the GNU GPL licence 2. </p> community@ez.no (Damien Pobel) http://projects.ez.no/ezvideoflv/news/ezvideoflv_datatype_0_2 Mon, 14 Jan 2008 16:24:55 +0000 First release of eZVIdeoFLV Datatype http://projects.ez.no/ezvideoflv/news/first_release_of_ezvideoflv_datatype <p>I released the first version of <a href="http://ez.no/developer/contribs/datatypes/video_flv_datatype" target="_self">eZVideoFLV</a>, a new extension providing a datatype to store, handle and convert to FLV video files.
