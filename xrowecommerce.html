xrowecommerce Björn Dieding@xrow.de stable 1.1 eZ Publish 4.3.0 and higher, PHP 5.3 and higher, MYSQL 5 and higher
A Solution out of extensions and functionalty to build a power full e-commerce webshop.

Includes:

  • New Product, Manufaturer and User Classes
  • Invoices
  • Coupons
  • Payment build-in
    • Pay by Invoice
    • Cash on Delivery
    • Advance Payment
    • Debit card ( Europe )
    • Credit card
  • Fraud decetion ( captcha security )
  • New Shopaccount Handler
    • Shipping and Billing Address
    • Security ( Captcha )
    • Live Tax ID validation
    • Further Additional Fields
  • Automatic Ordering / Recurring Orders
  • Google Analytics Integration
  • Inventory
  • Ordering from print catalog
  • Automated Shipping with API Integration UPS, USPS.
  • Refunding
  • Forecasting
  • Reporting
  • New Variation Datatype
  • Aftersale
  • Export
    • Google Product Search
  • Usebility
    • Better error messages

In Development

  • Bugfixes

In Plannig:

  • Tracking and Barcodes ( no date yet )
  • Export catalog to Indesign ( no date yet )

Install Steps:

1.) Download and extract eZ Publish (min. Version 4.3.0) with eZ Components.

2.) Modify settings/package.ini and add/replace

[RepositorySettings]
RemotePackagesIndexURL=http://packages.xrow.com/xrowecommerce/latest
RemotePackagesIndexURLBase=

3.) Run the eZ Publish installer as normal and select the xrow ecommerce site package.
