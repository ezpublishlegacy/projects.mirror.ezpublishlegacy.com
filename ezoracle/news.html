eZ Oracle 2.0.1 released

Gaetano Giunta

Tuesday 30 June 2009 14:51

The latest version of the "eZ Publish database extension for Oracle", also known as eZ Oracle,

  • adds the upgrade script for eZ Publish up to 4.1.3
  • corrects a couple of bugs, one of which prevented the installation using 'fully manual' mode

Please note:

the mechanism used to derive the names of sequences for tables that are created by extensions has been changed in this release (sequences are the oracle-equivalent of self-incrementing fields in mysql). Take care if you are upgrading from a previous version and have installed any extension that uses persistent objects based on custom tables.

eZ Oracle 1.6, 1.7 and 1.8 released

Gaetano Giunta

Monday 22 December 2008 16:48

I am happy to announce the eZ Oracle Christmas Release!

After more than a year of work, versions of the extension corresponding to eZ Publish 3.9, 3.10 and 4.0 are finally generally available.

Lots of bugs have been fixed and performance has been tweaked to a great extent, too.

Please make sure to read with care the release notes and install/upgrade instructions that come with each version.

In particular:

  • eZ Oracle 1.6 is considered a stop-gap measure, as there are known bugs in eZ Publish 3.9.5 running on Oracle, and version 3.9.6 has not been released (it will most likely never be). Please upgrade at least to eZ Publish 3.10.1 if you plan to use an Oracle database with eZ Publish 3.
  • The "eZ Publish clustering" mode requires some patches to the eZ Publish kernel files for versions 3.10.1 and 4.0.1. All the relevant patches are included within the eZ Oracle extension
