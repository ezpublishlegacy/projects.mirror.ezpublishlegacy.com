Provides SPEC files for various stuff needed for eZ Publish.

RHEL 5 and Centos 5 supported. Fedora and others might work too. Contributors welcome.

Install the eZ Publish RPMS over YUM

rpm --nosignature -i http://download.fedora.redhat.com/pub/epel/5/x86_64/epel-release-5-4.noarch.rpm 
rpm --nosignature -i http://packages.xrow.com/redhat/xrow-repo-1.0-1.noarch.rpm
yum install ezfind-solr
yum install ezpublish-ezodf

Howto use the SRPMS

Prerequisite

  • yum -y install rpm-build

Build & Install

  1. rpmbuild --rebuild ezpublish-packagename.src.rpm
  2. rpm -i /usr/src/redhat/RPMS/noarch/ezpublish-packagename.noarch.rpm

Removal

rpm -e ezpublish-packagename.noarch.rpm

Just Unpacking

rpm -i ezpublish-packagename.src.rpm

will unpack the contents of the rpm in the directories

  • /usr/src/redhat/SOURCES
  • /usr/src/redhat/SPECS

rpmbuild -ba /usr/src/redhat/SPECS/ezpublish-packagename.spec

will build a new RPM

Install OSC on Centos

  1. rpm -i m2crypto-0.19.1-1.src.rpm
  2. rpmbuild -ba /usr/src/redhat/SPECS/m2crypto.spec
