This project provides a simple soap search service solution

Description

This solution provides a simple soap webservice search of eZ Publish content through the eZ Search engine.

Requirements

  • Based on the Nusoap extension

Download

  • Download: bcsoapsearch.tar.gz
  • GitHub: https://github.com/brookinsconsulting/bcsoapsearch

Example Usage

In our first example we offered this search through soap feature to eZpedia.

The idea was to offer a web service which could be used to provide search of eZ content objects using the eZ search engine.

Since the service is a soap based webservice it was relatively simple to create software bindings to allow for the existing irc bot tronicon to offer search of this webservice to users in using irc chat software visiting the irc channel #ezpublish (on freenode.net irc) through the tronicon irc bot.
