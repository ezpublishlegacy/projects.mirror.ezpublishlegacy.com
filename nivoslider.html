Nivo Slider

Last updated: Wednesday 02 May 2012 19:28

UNIX name Owner Status Version Compatible with
nivoslider Stefano Gattuso beta 0.5 4.4 - 4.5
A Javascript slideshow

A Slide show based on Nivo Slider (http://nivo.dev7studios.com/) and jquery.

Features of NivoSlider

  • 16 unique transition effects
  • Simple clean & valid markup
  • Loads of settings to tweak
  • Built in directional and control navigation
  • Packed version only weighs 15kb
  • Keyboard Navigation
  • HTML Captions
  • Free to use and abuse under the MIT license

Features of Extension

  • Selectable Effect
  • Customizable effects
  • Customizable transition time
  • Customizable pause on slide time
  • ...

How to Install

Read the doc\INSTALL file

How to use

Create an Nivo Slider object and set all parameters you want ( ex. set captionOpacityto 0.8)

Create some Nivo Slider Image like child of the Nivo Slide object. You can fill ALT field if you want the slide have an caption.

Embed the Nivo Slider wherever you want.

FAQ

Can i modify the embedded box size?

YES. On image.ini there is a section called nivoslider, jsut change the attribute filters.

Compatibility

I tested only on 4.4 but i think it work from 4.0+
