Version 0.1 of the ezpublishbuilder script is available

Gaetano Giunta

Tuesday 25 October 2011 20:41

All it can build so far is the eZ Publish Community Project package suitable for the MS Web Installer tool, starting from an existing eZ Publish installation. An exciting first step! (note: no zip is available for download, read more for how to install this tool...)

Instructions for download and usage:  

Prerequisites:
 

  • php
  • git

Steps: (donâ€™t worry, this looks long but is smooth and easy ;-) )
 

  • move to a clean directory for the wpi build (recommended)
  • if you have never installed ezpublishbuilder:
    • get the ezpublishbuilder installer from projects.ez.no:

wget http://svn.projects.ez.no/ezpublishbuilder/stable/pakefile.php 

    • bootstrap ezpublishbuilder: run

php pakefile.php

  • if you have already  installed ezpublishbuilder in the past:
    • check that you are running the latest version

php pakefile.php tool-upgrade-check

    • clean the build directory

php pakefile.php clean

  • configure the options file  pake/options-ezpublish.yaml file with the correct build number. Do not change other options
  • run the init task:

php pakefile.php init

  • copy the full eZ Publish CP sources as built so far in build/ezpublish (nb: not in subdirectories)
  • run the build task:

php pakefile.php build --skip-init

  • run the dist task

php pakefile.php dist

  • check that in the dist folder you have two files: a big zip and and xml file
