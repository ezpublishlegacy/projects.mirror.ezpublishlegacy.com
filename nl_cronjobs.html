NL Cronjobs

Last updated: Thursday 28 July 2011 21:46

UNIX name Owner Status Version Compatible with
nl_cronjobs Nicolas Lescure stable 1.0 4.4+
Launch cronjobs manually in admin interface.

Sometimes, administrators should launch cronjobs. This extension add a tab in back office where they can choose which cronjob to launch (even your custom ones). Only active cronjobs are listed. All outputs are logged, and can be viewed in the admin interface.
