Subtree Location

Last updated: Saturday 09 April 2011 21:45

UNIX name Owner Status Version Compatible with
subtreelocation Jonathan Bouzekri beta 0.2 4.2+
Provide action to add locations to a subtree.

This extension add the button "Add subtree location" in the location sections of the node. When the contributor launch this action, He can add a location to the entire subtree. The structure of the subtree is preserved by the action.

This extension provides a php script to execute this operation directly in the command line or to schedule and run it in background thanks to the ezscriptmonitor extension.

Planned Evolution :

  • Add view between click on action button and browser screen to configure the "add subtree location" action like depth of the operation, class filter, do not locate root subtree element, etc ...
