Flashupload

Last updated: Monday 21 May 2007 22:43

UNIX name Owner Status Version Compatible with
flashupload STEVO + beta 1.0 3.9
A toolbar extension that provides an interface to upload multiple files with Flash, Javascript & HTML. It provides a toolbox, an extension to the Website Interface toolbar and can be used in custom templates.

Features

  • Uses SWFUpload R5
  • Uses eZContentUpload with upload.ini settings
  • Upload multiple files at once by ctrl/shift-selecting in dialog
  • No page reloads necessary
  • Works on all platforms/browsers with Javascript & Flash 8+
  • Degrades gracefully to normal HTML upload form if not

Installation

  • Read INSTALL file

Use

  • Override upload.ini to meet your needs
  • Set policies for creating and editing the right classes in the right places
  • Head to a node, open toolbar + upload stuff there or click new button on the left of the Website Interface toolbar.
