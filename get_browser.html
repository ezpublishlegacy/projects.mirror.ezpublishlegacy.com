Get / Detect Browser

Last updated: Wednesday 23 February 2011 15:28

UNIX name Owner Status Version Compatible with
get_browser Thiago Campos Viana stable 1.0 4.0+
Detects user browser name and version.

Usage:

{get_browser()|attribute(show,1)}

Returns:

Attribute     Type     Value
version     string     '3.6.13'
majorver     string     '3'
minorver     string     '6'
build     string     '13'
name     string     'firefox'
useragent     string     'Mozilla/5.0 (Windows; U; Windows NT 5.1; pt-BR; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13 GTB7.1 ( .NET CLR 3.5.30729; .NET4.0E)'

Install

1) Copy get_browser into 'extension' folder

2) Activate get_browser

In administrator user interface, click 'setup' tab->'extensions' menu, select 'get_browser', click button 'Apply Changes' OR in settings/override/site.ini.append.php, add acivation configuration under "ExtensionSettings" section:

[ExtensionSettings]
ActiveExtensions[]=get_browser

3) Regenerate autoloads
In administrator user interface, click 'setup' tab->'extensions' menu, click button 'Regenerate autoload arrays for extensions',
or in eZ Publish installation folder, run "php bin/php/ezpgenerateautoloads.php -e"

4) Clear cache
Clear INI and template caches. (from admin 'Setup' tab or commandline)
