Release 1.0.0 available

Brookins Consulting

Monday 22 June 2009 19:09

Brookins Consulting is proud to announce the first release of BC Post To View Parameters extension version 1.0.0 (tar.gz, tar.bz2, zip). Download it today!

Description

The BC Post To View Parameters extension provides custom module views for html form parameter conversion to view parameters and redirection to destination to url + view parameters.

About

This extension is a functional example of how to use html forms to filter page results using view parameters and custom redirection module view.

Just install and activate the extension and change your template html form to point to one of our four different custom modules.
