Hi,

I started using your extension with the migration of the projects on which I'm working to ezp 4.2. Now I added galleries section in one of my projects as in each gallery view full I implemented an ajax image slider which navigation is controlled through javascript functions. In these javascript functions I use ezjscore/run module in order to load the template with the desired content returned from the ajax response. This content includes images which are perfectly displayed in all browsers except Internet Explorers. Internet Explorers just don't show <img> tag content returned from ajax template on the page - images are hidden! In ezp 4.1 we were using ezcore for the same functionality, but there all ajax content was displayed correctly in all browsers.

Please if anyone came upon a similar problem, share your experience with me! I'm looking forward for your help! Thanks in advance!
