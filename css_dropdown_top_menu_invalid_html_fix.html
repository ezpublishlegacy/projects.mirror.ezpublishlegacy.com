CSS Dropdown Top menu invalid HTML fix

Last updated: Friday 09 November 2007 10:20

UNIX name Owner Status Version Compatible with
css_dropdown_top_menu_invalid_html_fix Andy Caiger planning N/A 3.92,3.93

Pages using the CSS Dropdown Top menu version 1.1 ( will fail XHTML
validation because some tags are not closed properly.
This fix is a full replacement for the flat_top.tpl template file which
addresses those issues.

Generally it is a matter of adding
{section-else}
to allow for the case where there is no third or fourth level
menu.
Some IE 7 directives were also deleted.
