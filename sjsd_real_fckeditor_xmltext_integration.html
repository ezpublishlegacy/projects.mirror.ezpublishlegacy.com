SJSD (real FCKeditor xmltext integration)

Last updated: Tuesday 08 January 2008 11:19

UNIX name Owner Status Version Compatible with
sjsd__real_fckeditor_xmltext_integration_ liu spider planning 0.3.1 3.8+

SJSD is the real FCKeditor integration for eZ publish: it generates
native eZ xmltext format, rather than xhtml. SJSD provides a fully
integrated, feature complete and extensible WYSIWYG open source
online editor for eZ publish. It is implemented in the form of a
FCKeditor plugin and an eZ publish extension and requires no
modification to FCKeditor/eZ publish core. SJSD uses AJAX to
communicate with eZ server.

SJSD provides a plugin for FCKeditor, so you can upgrade FCKeditor
easily.
SJSD is distributed as an extension for eZ publish.
Almost all the functionalities found in eZ OE are implemented in SJSD.

REQUIREMENTS:
eZ publish 3.8 or above ((3.7 is not supported anymore, unless someone
volunteer to work on this)
FCKeditor 2.2 (download from
INSTALL
1. unzip sjsd-0.3.0.zip under extension dir of ez publish
2. unzip the zip file of FCKeditor 2.2 under extension/sjsd/
3. enable the sjsd extension in your admin interface
4. clear all cache (yes all, not only template)
5. If your ezpublish installation is virtual host and rewrite rule is
on, make sure *.html/*.xml/*.js can be accessed directly
6. if you upgrade from previous version, make sure you clean your
browser cache before using the new one
that's it, enjoy.
