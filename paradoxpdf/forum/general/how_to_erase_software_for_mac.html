How to removing software from mac
http://www.umacos.com/images/cleanmac/cleanmac.jpg
You can removing software from mac by following steps
Instructions
Step 1
Confirm that the application you wish to remove does not have an uninstaller. If it does, run the uninstaller to properly remove the application.
Step 2
Open your "Applications" folder and find the application you wish to remove.
Step 3
Drag the application and any associated folders into the trash.
Step 4
Open your Home folder, then go into the Library sub-folder and open the Preferences folder to determine if there are preferences files that need to be removed. If you find files or folders with the same name as the application you want to remove, drag them to the trash.
Step 5
Open your Home folder, then go into the Library sub-folder and open the Application Support folder to determine if there are support files that need to be removed. If you find files or folders with the same name as the application you wish to erase, drag them to the trash.
Step 6
Drag the Application's icon off the dock with your mouse to remove it from the dock if it is present.
Step 7
Right-click on the Trash on the dock and select "Empty Trash" to fully remove the files you have dragged there.
