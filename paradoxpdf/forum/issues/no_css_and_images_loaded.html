Hellos Marius,

I recently had a smiliar problem using pardoxpdf with an SSL certificate on one of my ez websites.
I had to add the SSL certificate as trusted in the java keystore of my webserver to get the css and images working agin.
You can find a a lot of articles on the web explaining how to that, but I think it would be a good idea to add some details about this particular configuration in paradoxpdf documentation, even if it's actually not directly related to paradoxpdf, but rather to flyingsaucer and java.
