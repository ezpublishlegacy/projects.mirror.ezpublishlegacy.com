GroupDocs.Viewer for Java | PDF & Office Document Viewer

Last updated: Monday 08 September 2014 13:24

UNIX name Owner Status Version Compatible with
groupdocs_viewer_for_java Marketplace Marketplace team stable 1.0.0 eZ Publish Community Project 2014.05
Integrate the GroupDocs Java document viewer to display PDF, Microsoft Word, Excel, PowerPoint and over 50 other types of documents on any page within your eZ Publish website.

With this add-on, you can easily integrate the GroupDocs' Java document viewer library into your eZ Publish website. The library can render 50+ types of common business document formats on the web. Once installed, you will be able to embed PDF, Microsoft Office, Visio, AutoCAD and other types of documents right on your eZ Publish web-pages. You website visitors can then view the documents when browsing the website.

Convenient UI for Viewing Documents on the Web

GroupDocs.Viewer for Java library also has a user-friendly interface that can be embedded to a web-page along with documents. The UI help website visitors navigate large multi-page documents in a browser. While viewing documents through the interface, users can:

  • Either scroll documents or turn pages with a button, like slides
  • Jump straight to the needed page within the document
  • Preview and navigate pages with the help of thumbnails
  • Search for, select and copy text to the clipboard right from embedded documents
  • Zoom in/out of document pages
  • Print and download documents from browser

HTML5-based Document Rendering

GroupDocs.Viewer for Java library has two document rendering modes. The first rasterizes documents and renders text on a separate layer over the images. The second one converts documents to HTML + CSS + SVG. Both methods provides native text rendering (users can select/copy text right from documents embedded to a web-page). This also ensures crystal-clear font rendering. HTML5-based document rendering also provides cross-platform compatibility. Users can view documents embedded on you website from any device with an HTML5-complient web-browser. There is no need to install any plugins or software. No need for Adobe Reader to view PDF documents, for example.

Display Over 50 Types of Documents

GroupDocs.Viewer for Java library supports the following document formats:

  • Portable Document Format: PDF
  • Microsoft Word: DOC, DOCX, DOCM, DOT, DOTX, DOTM
  • Microsoft Excel: XLS, XLSX, XLSM, XLSB, XML
  • Microsoft PowerPoint: PPT, PPTX
  • Microsoft Outlook: MSG, EML
  • Microsoft Project: MPP, MPT
  • Microsoft Visio: VSD, VDX, VSS, VSX, VST, VTX, VSDX, VDW
  • OpenDocument Formats: ODT, OTT, ODS, ODP
  • Plain Text File: TXT
  • Rich Text Format: RTF
  • Comma-Separated Values: CSV
  • HyperText Markup Language: HTM, HTML, MHT, MHTML
  • XML Paper Specification: XPS
  • AutoCAD Drawing File Format: DXF
  • Image files: BMP, GIF, JPG, PNG, TIFF
  • Electronic publication: EPUB

Installing the Viewer

Before you proceed, please be aware that this add-on requires GroupDocs' Java document viewer library to be installed on your server. You can download a free evaluation copy of the library from the GroupDocs website:http://groupdocs.com/java/document-viewer-library
