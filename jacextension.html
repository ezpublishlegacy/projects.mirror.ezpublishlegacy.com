Installing:

  1. jacextension/sql/mysql/jacextenion_data_structure.sql - create new DB Table jacextension_data
  2. activating the extension
  3. update autoloads.php (only ez 4.x version)
    php bin/php/ezpgenerateautoloads.php -e -v
  4. check permission settings
  5. test URLs:
    http://.../modul1/list/3/4/param3/6/param4/8
    http://.../modul1/create
