eZ GoogleTranslate

Last updated: Wednesday 18 April 2012 11:38

UNIX name Owner Status Version Compatible with
ezgoogletranslate Gaetano Giunta beta 0.2 4.1
This extension provides templates for editing attribute content with an added "translate" button.
When clicked, the attribute content is translated by a call to the Google AJAX Language API

(more translation providers will be eventually integrated)
