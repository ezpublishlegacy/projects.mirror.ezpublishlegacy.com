Worldpay payment gateway (pubsvn)

Last updated: Friday 13 January 2006 12:16

UNIX name Owner Status Version Compatible with
worldpay_payment_gateway__pubsvn_ Paul Forsyth planning 2.0 3.2 and above.

This gateway allows you to use Worldpay as your payment mechanism within
your eZ publish site.

There are two versions of this plugin.
Version 1.0 works with eZ publish 3.2, 3.3 and 3.4 and is available via
pubsvn only:
Version 2.0 is for 3.5 and above. However, it has only been tested with 3.6
but should without problems:
The attached file contains version 2.0.
Full details are included within the file at 'worldpay/doc/manual.txt'.
