Janrain Auth is an implementation of Janrain (formerly RPX) authentication framework into eZ Publish.
It allows you to authenticate users from social networks such as Facebook, Google, Windows Live, Yahoo!...

Important note : This extension has a dependency on SQLI Import for its content API. So you'll need to install it before using Janrain Auth.
