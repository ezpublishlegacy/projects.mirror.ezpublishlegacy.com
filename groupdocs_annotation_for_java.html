GroupDocs.Annotation for Java

With this add-on, you can seamlessly integrate a Java version of the GroupDocs HTML5-based document & image annotation library into your eZPublish site.

Multi-Format HTML5-Based Document & Image Annotation Widget

GroupDocs.Annotation for Java is an HTML5-based document annotation library designed as a middleware to let developers enhance their Java apps or websites with the capability to display common types of documents and images along with a comprehensive set of annotation tools. With this add-on, eZPublish developers can seamlessly integrate the library into their sites.

Designed based on the client-server architecture, GroupDocs.Annotation for Java converts documents to web-compatible content (a combination of HTML, CSS and images) on the server side, streams it to clients and then renders in users’ web-browsers. The library comes with a web-based UI that can be customized with your own CSS and embedded to your site as a widget. The UI provides end users with a comprehensive set of tools for convenient navigation, reading and annotation of large multi-page documents straight on your site.

Features at a Glance:

1. Support for 50+ document and image file types. To name a few, supported file formats include: PDF and Microsoft Word documents, Excel spreadsheets, PowerPoint presentations, Outlook messages, Visio diagrams, OpenDocument files, CAD and raster images (TIFF, JPEG, GIF, PNG, BMP).

2. Cross-platform compatibility. End users can read and annotate documents from any web-enabled device and using any modern web-browser that supports HTML/CSS/JS. There is no need to install any office software or browser plugins. Supported browsers include: Internet Explorer 8+, Chrome, Mozilla Firefox, Safari 5+ and Opera.

3. A comprehensive set of annotation tools, including: sticky notes, area/point annotations, polyline, rectangle, arrow, text underline/strikethrough, text/area redaction, watermarks, ruler, etc.

4. Multi-user annotation sessions. GroupDocs.Annotation allows several users to annotate the same document or image simultaneously, each seeing others’ annotations and comments in real-time.

5. Support for native PDF & Microsoft Word annotations. PDF or Word documents annotated online using GroupDocs.Annotation can be saved along with the added annotations for further offline review and editing using original software.

6. Advanced document security. GroupDocs.Annotation doesn’t download source documents to clients during view/annotation sessions. Instead, the library renders only web copies of the original documents. End users don’t have access to your source documents, unless you allow them to. Download, print and copy options can be disabled for specific documents, so that they are shared in a read-only mode. Additionally, you can add custom watermarks over the displayed documents or completely hide sensitive data using the redaction tool.

Installing the Add-on

Please note! This add-on requires the Java version of the GroupDocs HTML5-based document annotation library to be installed on your server first. The library is not free, but we offer a free evaluation, so that you can test the annotator on your site. For more details on the library, available deployment options and to download an evaluation copy, please see this page.
