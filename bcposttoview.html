BC Post To View Parameters

Last updated: Tuesday 22 November 2011 04:44

UNIX name Owner Status Version Compatible with
bcposttoview Brookins Consulting stable 1.0.0 4.0+, 4.1+
Provides custom module views for html form parameter conversion to view parameters and redirection to destination to url + view parameters.

This extension is a functional example of how to use html forms to filter page results.
Brookins Consulting is proud to announce the release of the BC Post To View extension!
