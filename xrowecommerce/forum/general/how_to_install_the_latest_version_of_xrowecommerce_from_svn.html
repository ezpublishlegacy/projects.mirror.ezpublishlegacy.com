If you want to install the latest version of xrowecommerce, a quick way is to:

1. Download the latest code from the SVN repository
svn checkout http://svn.projects.ez.no/xrowecommerce


2. Make a new folder called "local" in the var/storage/packages/ directory in your eZ Publish install

3. Copy the contents of the xrowecommerce/packages directory into var/storage/packages/local

4. Run the eZ Publish setup wizard - it will detect and install these packages, as well as downloading any dependencies


If you want to build your own snapshot of xrowecommerce, you'll need to install Apache Ant, http://ant-contrib.sourceforge.net/tasks/tasks/propertycopy.html , and update build file settings to match local paths.

Xrowecommerce install on existing ezpublish site 2012.8 with ezflow extension (testsite)

Copy xrowecommerce into /extension directory

set permissions for /extension/xrowecommerce/settings

insert sql into database

install xrowecommerce_classes.ezpkg from extension/xrowecommerce/packages

activate extension xrowecommerce

activate webshop tab in admin top menu

add to /settings/siteaccess/site :

content.ini.append.php




    [VersionView]


AvailableSiteDesignList[]=xrowecommerce


 


design.ini.append.php

 


[StylesheetSettings]


CSSFileList[]=xrowecommerce.css


CSSFileList[]=../lib/yui/2.8.2/build/calendar/assets/calendar.css


CSSFileList[]=../lib/yui/2.8.2/build/calendar/assets/skins/sam/calendar-skin.css


 

[JavaScriptSettings]


JavaScriptList[]=ezjsc::yui3


JavaScriptList[]=ezjsc::yui3io


JavaScriptList[]=../lib/yui/2.8.2/build/yahoo-dom-event/yahoo-dom-event.js


JavaScriptList[]=../lib/yui/2.8.2/build/calendar/calendar-min.js


JavaScriptList[]=ezdatepicker.js


JavaScriptList[]=insertmedia.js


JavaScriptList[]=xrowecommerce.js


JavaScriptList[]=xrowproductvariation.js


override.ini.append.php




[xrow_client_edit]
