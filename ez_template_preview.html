eZ Template Preview

Last updated: Monday 04 June 2007 08:12

UNIX name Owner Status Version Compatible with
ez_template_preview Kåre Køhler Høvik planning 0.1a 3.8 / 3.9

Adds preview functionality of node/view/full.tpl template overrides.

Templates edited in the administration interface may now be previewed before
they are committed to the system.
Installation:
1. checkout/export the extension in the extension folder
2. Enable the extension in site.ini
3. Clear template override cache
Usage:
1. Enter the design panel in the administration interface.
2. Click "Templates" in the left menu.
3. Cick "/node/view/full.tpl" in the "Most common templates" list.
4. Edit one of the overrides.
5. Select site access to preview the changes in, and press
"Preview"
6. Select which node the content in the preview should be taken
from
7. Done, the system show the template with the changes made.
8-> Go back to edit or publish the template.
