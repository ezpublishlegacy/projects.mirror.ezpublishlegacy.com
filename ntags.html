nTags

Last updated: Wednesday 07 March 2012 09:36

UNIX name Owner Status Version Compatible with
ntags Leif Arne Storset stable 1.2 4.3
nTags extends eZ publish's keyword feature. Instead of a simple text box you get checkboxes for predefined tags (keywords), and an additional text box for free tags. You may also mass apply tags in content view.

This extension is suitable if you already use keywords or need a drop-in solution with predefined tags and mass tagging. If you're just starting out and want a more sophisticated solution, have a look at eZ Tags. (I haven't tried it myself, but it looks snazzy.)

This extension requires eZJSCore and is designed for the "admin2" interface. If you're stuck with the old-style admin interface, use the older version 1.1 of nTags.

Tagging GUI

  • Checkboxes for predefined tags and free-tagging field automatically fill in attribute field using JavaScript
  • Mass add/remove tags using AJAX GUI

GUI for editing predefined tags

  • Top menu tab
  • Add/delete predefined tags
  • Reorder using AJAX

Miscellaneous

  • Permissions for tagging and administration
