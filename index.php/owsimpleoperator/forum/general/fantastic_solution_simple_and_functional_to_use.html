<?php


 

class FirstTrueOperator extends OWSimpleOperator


{


    

   public function first_true( $value1, $value2, $value3, $value4, $value5, $value6, $value7, $value8, $value9, $value10 )


   {


       $arg_list = func_get_args();


                for ($i = 0; $i < func_num_args(); $i++) {


                        if ($arg_list[$i]) {


                                return $arg_list[$i];


                        } 

                }


   }


    

}


?>

Hi Reinhard!

Did you follow the Readme tutrial ? https://github.com/Open-Wide/OWSimpleOperator#step-1--install
In other words, did you register your operator ini your autoloads/eztemplateautoload.php extension file ?

By the way, you can easily simplify your code. 



<?php


class FirstTrueOperator extends OWSimpleOperator


{


    

  public function first_true( )


  {


     foreach( func_get_args() as $arg ) {


        if ( $arg ) {


           return $arg;


        } 

     }


  }


    

}


?>
