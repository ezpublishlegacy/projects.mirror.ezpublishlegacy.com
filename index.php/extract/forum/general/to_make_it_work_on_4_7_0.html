I've used this extension quite often on 4.x and I find it very useful.

Recently I installed it on 4.7, and it failes due to a call to a deprecated eZ php class:

"Class 'ezI18n' not found in /home/httpd/opencontent.glauco.it/html/extension/extract/modules/extract/csv.php on line 365"

To make it work, you ned to patch the file csv.php, and substitute the calls (there are two) to ezI18n with calls to ezpI18n::tr

the wrong bit of code:



$Result['path'] = array(   array('url' => false,


                                             'text' => ezI18n('design/standard/extract', 'Extract') ),


                                       array('url' => false,


                                             'text' => ezI18n('design/standard/extract', 'CSV') )


                                              );

the patched code:



$Result['path'] = array(   array('url' => false,


                                             'text' => ezpI18n::tr('design/standard/extract', 'Extract') ),


                                       array('url' => false,


                                             'text' => ezpI18n::tr('design/standard/extract', 'CSV') )


                                              );


This is similar to what is explained here for some other extension:
http://share.ez.no/forums/extensions/ez-classlists-and-ez-4.5

Hope it helps other people,
Mirko.
