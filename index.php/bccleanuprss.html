About

RSS Cleanup Extension for eZ Publish. Cleans up rss import cache within eZ Publish

Description

Provides eZ Publish Script, Cronjob and Settings.

Use either PHP Cli Script or eZ Publish Cronjob to clear rss cache within your eZ Publish website.
