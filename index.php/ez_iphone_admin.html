eZ iPhone admin

Last updated: Wednesday 17 March 2010 13:05

UNIX name Owner Status Version Compatible with
ez_iphone_admin Lars Bratli planning 1.0 4.1+

The eZ iPhone admin (vzthhadmin) extension allows you to easy manage
content
with your iPhone or other mobile device.

The extension delivers an iPhone-adjusted siteaccess where you can login
from anywhere to perform the simplest of CM tools. It loads as little
markup as possible, making it faster (and cheaper) when connected via
3G, EDGE GPRS etc.
Supported content managing tools are:
- new objects
- editing
- removal
- moving
- translation tools
- search
It also enables the admin user to clear caches.
NOTE that it will (probably) work on most other mobile devices as well. I
have not done extensive testing on 4.0, but is hopeful it is
working.
