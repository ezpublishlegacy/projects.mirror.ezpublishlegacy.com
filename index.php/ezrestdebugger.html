eZ Publish REST debugger

Last updated: Sunday 20 March 2011 00:00

UNIX name Owner Status Version Compatible with
ezrestdebugger Jérôme Vieilledent beta 1.0 4.5+
Debugger application for eZ Publish REST interface

This autonomous application lets you debug your REST services developed on top of eZ Publish REST interface (requires eZ Publish Matterhorn). It supports both OAuth 2.0 and Basic Auth authentication styles.

eZRestDebugger is multi-platform (Mac, Linux, Windows) and runs on top of Adobe AIR runtime.

Source can be found on GitHub.
