eZ Extension Builder

Last updated: Sunday 01 April 2012 18:00

UNIX name Owner Status Version Compatible with
ezextensionbuilder Jérôme Renard beta 0.4 4.x
Provides tools to build (package) extensions for eZ Publish.

This is NOT an extension generator - it provides scripts to help packaging extensions for distribution.

Two versions are available:

  • one based on Pake
  • one based on ANT - currently unmaintained - available at http://svn.projects.ez.no/ezextensionbuilder/
