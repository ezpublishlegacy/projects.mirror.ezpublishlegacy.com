Hi,
I got it working now.

Fixed few typos:

block_item/nivoslider.tp

in line 38 "Double quote" is missing and replaced "nivoslider_embed" with
"nivoslide_block_item"


and in block/nivoslider.tpl put this code:



    {def $valid_nodes = $block.valid_nodes}


 

<!-- BLOCK: START -->


 

{def $my_node=fetch( 'content', 'node', hash( 'node_id', 154 ) )}


{node_view_gui view='block_item' content_node=$my_node}


 

 

<!-- BLOCK: END -->


 

{undef $valid_nodes}   

 


with known node ID so I could check if it will work.
