suit for mainland china, tested with ez3 subversion 2.

I've done some job on ez3 Chinese translation.
that's it. china20030502.zip
although it's partly finish (I've just tranlated a few admin interface), it's
work.
contan two files:
chi-CN.ini should place it to folder share/locale/
translations.ts should place it to folder share/translations/
change file settings/site.ini or other file like
settings/override/site.ini.append...
[RegionalSettings]
Locale=chi-CN
ContentObjectLocale=eng-GB
TextTranslation=enabled
TranslationCache=enabled
change file settings/i18n.ini or other file like
settings/override/i18n.ini.append...
[CharacterSettings]
Charset=UTF-8
note: chi-CN is just suit for Mainland China. Hongkong and Taiwan are
defferent charsets
so can't use this tanslation.
for any further question about it. please e-mail or
a better chi-CN translation zip file will come soon:)
