States check

Last updated: Tuesday 24 November 2009 02:02

UNIX name Owner Status Version Compatible with
ezstatescheck Eric Sagnes stable 1.0 4.2
An extension for checking the states of the objects that are in a state group.

By using this extension, users can check the status of objects in a state groups they are part of.

It basically add a new button to the Web Site toolbar.

  • If the button is red, it means the current user can change the status of one of the items in the group. In shorter terms, the user has a pending item.
  • if it is grey, it means that there are objects in a state group the user can participate, but cant change the state of the objects. In shorter terms, the user can check the states of objects after or before his step.
  • If there is no editable and uneditable objects, the icon will not show up.

It can also be accessed directly by states_check/view
