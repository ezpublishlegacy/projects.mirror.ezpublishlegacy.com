ezosm

Last updated: Friday 16 April 2010 14:02

UNIX name Owner Status Version Compatible with
ezosm Sebastian Schoeller alpha 0.1 4.x
Simple overwrite for view provided by eZGmapLocation in order to use Openstreetmap data via OpenLayers API. Also an eZFlow block is provided similar to geo_located_content.tpl.

To install:

  1. Extract tar to extension directory
  2. Place ActiveExtensions[]=ezosm in settings/override/site.ini.append.php in front of ezgmaplocation and ezflow
  3. Clear all caches

Apparently the background layer can be switched through OpenLayers, so one could also utilize other e.g. NASA imagery. The same counts for marker images.

I did this as a showcase. It serves my needs and I will probably not continue adding anything, if somebody wants to engage please do so.

Best wishes

Sebastian
