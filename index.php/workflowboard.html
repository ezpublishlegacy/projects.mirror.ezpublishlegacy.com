eZ workflowBoard

Last updated: Monday 28 December 2009 11:00

UNIX name Owner Status Version Compatible with
workflowboard J-A Eberhard stable 1.0 4.1 and above
WorkflowBoard extension add a new tab in the admin interface to allows you:
- To monitoor the progress of multi steps approval workfows
- To interrupt and cancel a running workflows

README WorkflowBoard
--------------------------
DESCRIPTION
It is not yet easy in monitoor, or interrupt, a workflow in eZ publish.
This extension provide you with a new tab in the administration interface of eZ publish.
In this tab you will be able to;
- See, step by step, the workflow progress. If a workflow has several steps, the step-by-step progress is displayed allowing you to see where the workflow is bloqued.
- Cancel a worflow at any time, the content objet is returned as a draft for the author.
--------------------------
CREDITS
This extension is based on the work by Smile for the smileworkflowboard for eZ 3.6
--------------------------
INSTALL
- Copy the workflowboard directory in your eZ publish extension directory
- Go to admin / setup / extension, and activate the extension
- Please clear cache ...
- Use it.
--------------------------
REQUIREMENTS
ezPublish 4.1 or above
