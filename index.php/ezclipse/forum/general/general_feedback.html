- Copy the update site url on the eZcliipse project page
- Go to Help > Install new software
- Click add, give the update site a name and paste the url
- Uncheck the "Group items by category" checkbox
- Now the eZclipse plugin shows up in the list. Select it by checking the box and go on with install.
