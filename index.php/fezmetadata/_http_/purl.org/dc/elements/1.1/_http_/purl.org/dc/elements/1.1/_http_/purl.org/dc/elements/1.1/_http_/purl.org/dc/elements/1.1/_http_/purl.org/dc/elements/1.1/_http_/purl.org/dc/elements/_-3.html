feZ Meta Data

Last updated: Tuesday 25 August 2009 11:27

UNIX name Owner Status Version Compatible with
fezmetadata Frédéric DAVID beta 1.0.0 4.0.x
Provides an interface to manage meta data. You can add, update and delete meta data for ez publish objects.

This extension provides an interface to manage meta data, like keywords or description.

Features

  • Manage Meta Data
  • Display Meta Data on the front site.

Requirements

  • eZ publish 4.0.x

Installation

  1. Download the archive
  2. Unpack it in the extension directory of eZ Publish
  3. Execute schema.sql script, for example from the eZ publish root :
    mysql -u user database -p < extension/fezmetadata/sql/schema.sql
  4. Activate the extension with the Administration interface in System > Extensions
  5. Regenerate the extension autoloads
  6. Clear the caches

Configuration

To add an available meta data, add a line in the AvailableMetaData table in settings/ezmetadata.ini.append.php
