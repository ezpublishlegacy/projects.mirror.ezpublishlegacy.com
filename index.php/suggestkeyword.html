SuggestKeyword

Last updated: Tuesday 01 September 2009 19:29

UNIX name Owner Status Version Compatible with
suggestkeyword *- pike planning 0.7 4.0.3

This extension overrides the datatype/edit template of the ezkeywords
datatype, changing the default textbox into a widget similar to the
locationbar in explorer, or the mailaddress bar in most mailclients.

--------------------------
I've updated this for ezp4.0, but you probably should consider this
instead:
--------------------------
The usual textbox is replaced by a textbox that,
when you type in it, folds out to show keywords
that contain the characters you've typed after
the last comma. You can use the mouse or arrowkeys
to select one of those entries, or ignore it.
It is build upon the SuggestFramework, from
which uses json to show its suggestions. The
source files have been adjusted a bit for
the specific use in this extension, and one
json view has been added.
I would not call this a "hack".
But it is a "alternative solution than something
which is included in eZ publish", which is the
description of the hack contribs here ...
--------------------------
CONTENTS
Essentially, these files do all the work:
modules/suggestkeyword/jsonkeywords.php
design/standard/templates/content/datatype/edit/ezkeywords.tpl
design/standard/stylesheets/suggestkeyword.css
design/standard/javascript/suggestkeyword.js

--------------------------
INSTALL
- Copy the suggestkeyword directory in your eZ publish extension
directory
- Go to admin / setup / extension, and activate the extension
- Please clear cache ...
- Edit a node that has an attribute of datatype 'ezkeywords'
- When typing in the keyword box, notice the popup that folds out

- Infinite joy.
--------------------------
FINETUNING
This should work fine 'out of the box', but you
may want to finetune things, eg. in your own designs.
Here's how.
To use this in your own design, try to override
the datatype/edit widget by placing a copy in
extension/suggestkeyword/design/yourdesign/templates/content/datatype/edit/ezkeyword.tpl
You can change some settings in that template.
Amongst others:
-action
The 'action' attribute is set to
a limit of 10 results for the json call
-delay
The SuggestFramework uses a 'polling'
mechanism that repeats timeouts to see
if the textbox has changed. You can set
the speed of the timeout on the textbox
-treshold
I've added a treshold of 3 characters
before it starts showing suggestions.
You can change that number there as
well.

REQUIREMENTS
ezPublish 3.10>, php4.
Tested on 3.10.1,
on Firefox/Mac and Safari 10.3.9
Let me know if this works for you !
