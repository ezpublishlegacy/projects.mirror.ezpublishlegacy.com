The package of the Enhanced eZBinaryFile extension available for download within this project's downloads is the latest version compatible with eZ Publish 4.x <= 4.6 (Tested with eZ Publish 4.5)

If you wish to use the Enhanced eZBinaryFile extension for eZ Publish 3.x through 3.10.x then please download the older version which is eZ Publish 3.x+ compatible directly from the older version's project website.

Remember this extension has a separate project for the eZ Publish 3.x+ version through it the eZ Publish 3.x+ compatible version is available, http://projects.ez.no/enhancedezbinaryfile

If you wish to use the Enhanced eZBinaryFile extension for eZ Publish 4.x+ then please download the latest version from this project's downloads or subversion repository!
http://projects.ez.no/enhancedezbinaryfile/downloads

Worth noting is that the project for the eZ Publish 3.x version is clearly not maintained any longer (also it is a project imported from the older ez.no contributions section, and these seem to have remained unmaintained since long before the migration).

I hope this helps other users looking to obtain the features this extension provides in a very simple to use all in one solution.
