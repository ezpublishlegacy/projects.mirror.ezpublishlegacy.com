eZ Rest

Last updated: Thursday 25 June 2009 09:04

UNIX name Owner Status Version Compatible with
ezrest Vidar Langseid beta 1.0.0beta1 3.x and 4.0+
REST framework for eZ Publish 4.0+ . For more information about REST: http://en.wikipedia.org/wiki/Representational_State_Transfer

eZ Rest is an extension providing an easy to use framework for creating REST web services with eZ Publish.

The eZ Rest extension is used by upcoming versions of eZ Find ( 1.1 ).
