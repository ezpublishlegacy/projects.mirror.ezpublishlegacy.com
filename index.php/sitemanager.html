Sitemanager

Last updated: Wednesday 21 October 2009 14:44

UNIX name Owner Status Version Compatible with
sitemanager S V planning 0.9 3.5

The Sitemanager extension offers an admin interface to create, modify,
enable/disable and remove eZ publish site accesses. It can be used to
create site accesses from standard, predefined 'packages'.

The package manager of eZ publish 3.5 does not yet support the creation
and import of complete site access packages. The Sitemanager
extension offers a temporary solution in the form of an admin
interface to create, modify, enable/disable and remove complete eZ
publish site accesses.
With the sitemanager it is possible to:
- create site accesses from standard, predefined 'packages', such as
standard extranet sites for projects;
- enable/disable or remove site accesses; and
- view siteaccess attributes, such as the size of the storage and
cache directories.
The extension has the following limitations and bugs:
- The site 'packages' does not follow the eZ publish standards of
packaging;
- The sitemanager does not copy 'addon' and 'style' packages;
- The sitemanager may not fit everyone's file/database permission
policies;
