eZ FTP

Last updated: Friday 05 June 2009 14:04

UNIX name Owner Status Version Compatible with
ezftp Damien Pitard alpha 0.0.2 4.0.x - 4.1.x
Give your eZPublish the ability to act as an FTP server, enabling you to download and upload content (image, article...) from any FTP client (Filezilla, Fetch) or directly from a mounted remote drive in your file browser (with fuse).

Could be a good alternative to webdav...

Download the latest release

  • release 0.0.2 (unstable)

You should better check out sources from subversion, until a stable version has been released.

Subversion

You can check out the latest source code of eZFTP released from its Subversion repository:

svn checkout http://svn.projects.ez.no/ezftp/trunk

Work with (tested)

  • FileZilla 2.2.x (Windows XP)
  • Fetch 5.0.x (OSX)
  • Transmit (OSX)

Requirements

The following requirements exists for using eZ FTP extension:

  • Operating system:

Linux (tested on Debian Etch).
You need to launch the daemon as root if you want to use standard port (ports number 20, 21)
This extension will not work on Windows system.

  • eZ publish version:

Make sure you use eZ publish version 4.0 or higher.

  • PHP version:

As indicated in your eZ publish version requirements.
PHP set with sockets, and posix support.
