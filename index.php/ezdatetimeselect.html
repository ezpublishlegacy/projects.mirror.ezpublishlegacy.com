ezDateTimeSelect

Introduction:
The ezDateTimeSelect extension is a popup calendar for ezdate and
ezdatetime attributes. It places a button next to ezdate and
eadatetime attributes which allows the user to select the date or
date and time from the popup calendar. It was created using the
Dynarch DHTML Calendar, which is free software published under the
terms of the GNU Lesser General Public License.
For the entire license text please refer to
Installation instructions:
- Unzip the downloaded extension to the 'extension' folder in your
ezPublish root directory
- Enable the extension in using the Admin interface (setup tab,
extensions link)
Options:
The Dynarch DHTML calendar comes with a number of different css files to
change the look of the calendar. You can change which css file is
used by editing the design.ini.append.php file in the extension's
settings directory. For example:
Before (theme.css):
[StylesheetSettings]
CSSFileList[]=skins/aqua/theme.css
# CSSFileList[]=calendar-blue.css
# CSSFileList[]=calendar-blue.css
...
After (blue.css):
[StylesheetSettings]
# CSSFileList[]=skins/aqua/theme.css
CSSFileList[]=calendar-blue.css
# CSSFileList[]=calendar-blue.css
...
Also, if you look at the Dynarch DHTML Calendar reference.pdf document
you'll see that there are more fancy things that you can do with the
calendar.
Credits:
