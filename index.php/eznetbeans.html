eZNetbeans

Last updated: Tuesday 31 May 2011 21:51

UNIX name Owner Status Version Compatible with
eznetbeans Thiago Campos Viana stable 0.3.3 4.4+
eZNetbeans is a Netbeans plug-in for supporting the eZ Publish tpl files.

Version 0.3.3 and source available at http://projects.ez.no/eznetbeans/downloads

To use the palette, drag and drop the palette item to source editor.

Please restart the IDE after installing the module!

Basic syntax highlighting, netbeans 7.0 opens tpl as xhtml files and ts as xml files.

0.2.1 - Started palette feature - drag and drop items to use.

0.2.2 - Array, images, misc and string palette items.

0.3.0 - Palette with all default template operators+ezjscore operators

0.3.2 - Basic code completion.

0.3.3 - Added a clear cache button ( main project only ).
