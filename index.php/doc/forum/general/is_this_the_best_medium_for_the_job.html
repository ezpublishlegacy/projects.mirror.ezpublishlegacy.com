@ André R.

Thank you Andre!

Per your request I have made the following modifications.

1. Linked to the forum topic in the forum template (this way the discussion menu link remains displayed even when within the forum / forum topic).

2. Description ... I'm considering your request for a description / intro of the wiki entry. I'm not sure how best to implement offhand since the wiki articles do not have a short description / intro field within the entry.

For now I added a link within the forum template to display the name of the wiki article with return link.

3. Thread message display order - I have also reversed the default order to display messages (as most recent) first to help avoid unnecessary scrolling or pagination and improve usability.

4. Latest updates rss feed - I have added an rss feed with only the latest updated conteent objects on ezpedia. http://ezpedia.org/rss/updated/discussion

5. Latest discussion (only) updates rss feed - I have added an rss feed with only the latest updated content objects on ezpedia. http://ezpedia.org/rss/updated/discussion

6. Homepage - Layout - I have also added a list of recent discussions into the homepage layout. http://ezpedia.org/
