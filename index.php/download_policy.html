Download policy

Last updated: Friday 09 March 2007 22:18

UNIX name Owner Status Version Compatible with
download_policy Patrice Fortin planning 0.1 3.9.x

This is a download handler. Just before downloading a file, it checks if
the content object for the file has the attribute policy set to true.
If so, the user is prompted a download policy which must be accepted
in order to download the file.

The handler code is based on ezfilepasstroughhandler.php and
ezbinaryfilehandler.php
To install extension:
1- activate extension from admin interface
2- modify the file class by adding a field 'policy' of type
'checkbox'
3- create in your site an article for the policy that must be read and
another article saying that the user cannot download file since he
didn't accept the policy
4- set RefusedPolicyNode and ReadPolicyNode in
ezacceptpolicy/settings/site.ini.append.php to the nodeID of created
articles in 3 (or comment it in
ezacceptpolicy/settings/site.ini.append.php and add it in
siteaccess/your_site/site.ini.append.php)
