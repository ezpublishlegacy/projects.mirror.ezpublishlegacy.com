This extension now works with eZ Publish Community Release 2014.11 (Legacy Stack) aka eZ Publish 5.x. Note that this version also works with eZ Publish 4.x as well. Get the latest version of the bcuserregisteruserplacement extension (formerly known as placeuser or Automated user placement) updated to work with eZ Publish 5.x Legacy Stack and re-implemented to improve the solution in many ways. Now available on GitHub today! https://github.com/brookinsconsulting/bcuserregisteruserplacement Anyone is free to use (clone) this repository, just watch it for changes, or fork this new repository and use it as needed freely. Run the following command from your project root to install the extension: <code>bash$ composer require brookinsconsulting/bcuserregisteruserplacement dev-master;</code> Alternatively you can download the latest source code from github in an automatically created zip file: https://github.com/brookinsconsulting/bcuserregisteruserplacement/archive/master.zip Cheers, Brookins Consulting community@ez.no (Brookins Consulting) http://projects.ez.no/automated_user_placement_for_ez_publish_3_8/forum/general/automated_user_placement_now_works_with_ez_publish_5_x Sun, 14 Jun 2015 02:51:39 +0000 Changelog http://projects.ez.no/automated_user_placement_for_ez_publish_3_8/news/changelog <p> 2.0<br /> - release for eZ publish 3.8+</p> community@ez.no (Kristof Coomans) http://projects.ez.no/automated_user_placement_for_ez_publish_3_8/news/changelog Tue, 12 May 2009 20:21:09 +0000 placeusers.tar.gz http://projects.ez.no/automated_user_placement_for_ez_publish_3_8/downloads/placeusers_tar_gz community@ez.no (Anonymous User) http://projects.ez.no/automated_user_placement_for_ez_publish_3_8/downloads/placeusers_tar_gz Tue, 12 May 2009 20:21:09 +0000
