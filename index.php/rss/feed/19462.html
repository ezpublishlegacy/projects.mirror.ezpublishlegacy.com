But we have often the use case to only have a cropped version with a special aspect ration for example in lineviews or for gallerythumbnails and for the full view the original aspect ratio. For this usecase we have a crop extension. We define which aspect ratio we want for this a class_image. The editor can't choose the aspect ratio (but it could be possible). We only store the crop information for the original image and have a new crop filter for image.ini. Then we can choose on which imagealias we want to use the crop or not.
