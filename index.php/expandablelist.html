expandablelist

Last updated: Saturday 14 January 2006 10:55

UNIX name Owner Status Version Compatible with
expandablelist Fraser Hore planning .01 3.7

This extension provides an ez way to include an expandable list of tree
nodes in any template. It uses javascript and css from David
Lindquist at gazingus.org.

This extension provides an ez way to include an expandable list of tree
nodes in any template. It uses javascript and css from David
Lindquist at gazingus.org (.
Installation
---------------
1) Unzip the extension and copy it to your extension folder
2) Enable the extension in the ez admin interface
Use
------
Copy the following lines into any template:
{def $expandable_rootnode_id=$node.node_id}
{def $expandable_style_class=imgswitch}
{include uri="design:expandablelist.tpl"}
Notice that the code above defines a root node by assigning a node id to
$expandable_rootnode_id and a style by assigning a style class to
$expandable_style_class. You can assign any root node that you like.
There are three available style classes in the stylesheet: simple,
switchalignedright and imgswitch. You can easily edit the stylesheet
to change the look of your list.
