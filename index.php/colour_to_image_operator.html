Colour to Image Operator

Last updated: Wednesday 29 October 2008 09:19

UNIX name Owner Status Version Compatible with
colour_to_image_operator Nathan Kelly planning 1.0 4.0.x

Creates a new image from supplied rgb or hexadecimal arguments and stores it
if it doesn't already exist. Returns the path to the image that was
stored.
Uses eZImageInterface (/lib/ezimage/classes/ezimageinterface.php) for
image creation.

Usage
rgbtoimage( r, g, b, [options] )
hextoimage( 'hex', [options] )
