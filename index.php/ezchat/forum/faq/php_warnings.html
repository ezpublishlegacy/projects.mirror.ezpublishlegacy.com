Hi,

Sorry for this response, but this extension was written for eZ Publish 4.0.0, and does not support PHP5 at all.

To work fine, this extension should be rewritten from scratch in PHP5. The latest eZ Publish 4.X is very different from the old 4.0.0, so you could encounter a lot of problems.

Since I'm not on the eZ Publish scene for more than 3 years, I cannot do this by now.
