Open Social

Last updated: Wednesday 31 December 2008 10:45

UNIX name Owner Status Version Compatible with
opensocial Paul Forsyth planning N/A N/A
This project will examine how best to integrate and use Google's OpenSocial API.

Information about OpenSocial is found here:

http://code.google.com/apis/opensocial/

The project is being planned. This means that the requirements for the project have still to be defined. Everyone is welcome to pitch in their ideas!

I see two distinct deliveries:

  1. Read from other social networks and display this data within eZ pages.
  2. Allow other social networks to view data derived from eZ users and applications.

The ' data' can be anything, from simple user information to applications.

More soon.
