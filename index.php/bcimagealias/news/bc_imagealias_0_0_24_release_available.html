BC ImageAlias 0.0.24 release available

Brookins Consulting

Monday 31 October 2011 05:12

Brookins Consulting is proud to announce the release of the BC ImageAlias extension version 0.0.24 (tar.gz, zip, tar.bz2, ezpkg, rar, 7z) Download it today!

BC ImageAlias provides a perfect solution for quick and simple generation or removal of content object image alias variation image files before they are needed or requested by the users of your site.

The 0.0.24 release features further advancements in documentation provided within the extension itself. Simply update the extension to 0.0.24 and use the extension normally.

This release's primary development was contributed by Brookins Consulting!

The new version is now available for download or GitHub bcimagealias repository

Description

This extension provides for easy to use generation and removal of content object image alias image variation image files.

Just install and activate the extension and you can use the command line script, cronjob part or workflow event to generate content object image alias image variation image files before they are requested by the users of the site.

Past versions

The 0.0.23 release features a large amount of new features (new parameters, new module views, new menu templates, new translations, new scripts, new cronjobs, workflow event improvements, general improvements and bugfixes. Simply update the extension to 0.0.23 and use the extension normally.

The 0.0.22 release features a large amount of critical functional bugfixes to ensure image alias handler records of generated and removed alias entries are stored properly. Simply update the extension to 0.0.22 and use the extension normally.

The 0.0.21 release features a critical bugfix to ensure image alias handler records of generated and removed alias entries are stored properly. Simply update the extension to 0.0.21 and run extension/bcimagealias/bin/php/bcimagealias.php --generate and then extension/bcimagealias/bin/php/bcimagealias.php --remove again to restore this data to an accurate state. Apologies for any inconvenience.

The 0.0.20 release featured a large number of improvements including ...

  • Tested entire extension to ensure proper support of eZ Publish 4.5+ (2011.09)
  • Re-implemented command line script to provide even more useful functionality
  • Added php class to generically provide required functionality in a reusable and flexible way
  • Added command line script option to remove all image alias variation image files
  • Added cronjob part to generate image alias variation image files
