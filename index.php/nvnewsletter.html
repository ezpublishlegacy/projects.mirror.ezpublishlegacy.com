NV Newsletter

Last updated: Thursday 10 March 2011 14:11

UNIX name Owner Status Version Compatible with
nvnewsletter Tuomo Syvänperä beta 0.8.1b 4.x
Newsletter extension for eZ Publish.

NOTE The svn repository here in projects.ez.no is no longer being maintained. Please use the provided zip - file in downloads -section.

Check out doc - folder for readme and install instructions.
Check out the changelogs here.

FEATURES

  • Subscribe/unsubscribe newsletter
  • Multiple newsletter receiver groups
  • Multiple sender info for newsletters (each newsletter can have it's own sender information ie. from - field)
  • Personalized newsletters. Receivers can have additional fields defined (first name, last name etc.), which are automatically replaced with correct values in newsletters which contain [[field]] - tags
  • Newsletter link tracking (clicks etc.)
  • Newsletter preview (in browser & sending a preview mail)

CHANGES IN 0.8.1

  • Workaround for a bug in postfix which broke html mails with (at least) MS Exchange servers

CHANGES IN 0.8

  • More detailed tracking of links etc. (check doc/README for instructions and limitations)
  • Automated installation using ezxmlinstaller (check doc/INSTALL for instructions)
