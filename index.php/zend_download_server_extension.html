Zend Download Server Extension

Last updated: Thursday 23 April 2009 09:30

UNIX name Owner Status Version Compatible with
zend_download_server_extension Björn Köster planning 2.5.1 4.1

The Zend Download Server Extension integrates the Zend Platform Download
Server in eZ Publish.

The new release of Zend Downloadserver is expanded by the support for
downloading image alias.
To realize the download of a scaled version of an image the
binaryhandler-function handleFileDownload was tuned.
In this function the path to the demanded image (which is stored in
var-directory) is detected by using the auxiliary function
splitFilename.
To make use of downloading an image the download link requires a
certain form:
.../content/download///
e.g.: .../content/download/123/234/my_image_small.jpg
Attention should be paid to the .htaccess-file: The access to var-directory
has to be regulated.
1. Description
The ZendDownloadServer Extension integrates the Zend Platform Download
Server (Version 1.5) in eZ Publish.
With the help of Zend Platform Download Server file-downloads can be
handled. Therefore a temporary
symlink is created. By minimizing webserver-processes during downloads
performance is improved.
ZendDownloadServer Extension is tested with eZ Publish greater 4.0.x
This Version of ZendDownloadServer Extension is not compatible to older
version of ZDS.
Upgrade Zend Platform to version 3.5
2. Installation + configuration
Please read install.txt for installation and configuration instructions.
