This extension provides a WYSIWIG editor ( for editing rich text fields
(ezxmltext fields).
It allows editors to beautify their content by handling html formatted
text. Based on an initial contribution by Geraint Edwards (, it
places the output between 

 tags. As it is not "normal" ezpublish format system, you should use this extension only if you know what it will imply for you. It includes the last release of FCKEditor (v2.2) and has a re-engineered integration. Only the FCK editing window is thus shown to the user, it now comes easy-to-use for the end-user. To suit my needs, I also added a specific toolbar settings mecanism based on node hierarchy. You can thus easily give a specific siteaccess configuration or specific settings to a node and all its children by specifying it in the siteaccess config file. It can also handle base configuration and external FCK config files, what can be useful in some contexts. Images and flash from media library can now also be browsed and uploaded on the fly with FCKeditor browser. Check the readme files for further informations on how to install and configure. Note: The latest release is always available on pubsvn ( ) Feel free to use and improve it!
