AutoPlayCarousel

Last updated: Thursday 30 September 2010 21:17

UNIX name Owner Status Version Compatible with
autoplaycarousel Thiago Campos Viana alpha 1.0 4.3+ with eZ flow
It's a new block type for eZ flow sites. It's like default media carousel block type, but has auto play.

Installation
================
1) Copy autoplaycarousel into 'extension' folder

2) Activate autoplaycarousel
In administrator user interface, click 'setup' tab->'extensions' menu, select 'autoplaycarousel', click button 'Apply Changes'.
or in settings/override/site.ini.append.php, add acivation configuration under "ExtensionSettings" section:
[ExtensionSettings]
ActiveExtensions[]=autoplaycarousel
3) Regenerate autoloads
In administrator user interface, click 'setup' tab->'extensions' menu, select 'autoplaycarousel', click button 'Regenerate autoload arrays for extensions',
or in eZ Publish installation folder, run "php bin/php/ezpgenerateautoloads.php -e"

4) Clear cache
Clear INI and template caches. (from admin 'Setup' tab or commandline)
