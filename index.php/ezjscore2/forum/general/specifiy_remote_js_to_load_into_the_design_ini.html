That is not correct.
You can not use ezjscore.ini for this kind of stuff, ezjscore.ini is for library loading and needs some pices of php code to work.

You can however use design.ini to load external scripts / css as long as ezjscore functions are used for loading JavaScriptList and so on.
But the url your mentioning in your example is a css file so will never work as a JavaScript no matter how hard you try :)
