Hello,
indeed, paradoxpdf.php is bugged on line 117 : 



$size = $handler->size();


$mtime = $handler->mtime();


If cache is empty $handler->size() and $handler->mtime() are null. If cache is already created, it's ok, the handler gets all information.

Here is my fix because the previous from Thierry Clément does not get the right size when cache is already created :




// if no cache


if ($data instanceof eZClusterFileFailure || $handler->size () == 0) {


   $data = $this->generatePDF ( $xhtml );


                                


   // check if error occurred during pdf generation


   if ($data === false) {


       return;


   }


 

   $handler->storeCache ( array ('scope' => 'template-block', 'binarydata' => $data ) );


   $size = strlen ( $data );


}


// if cache


else {


   $size = $handler->size();


}
