Great,


Suppose your content is structured like this



<div class="cover">


     <h1>cover page</h1>


     <!-- some other content here -->


</div> 

 

<div class="toc">


     <h1>Table of content</h1>


       <!-- some other content here -->


</div>


<div class="content">


     <h1>here is the content</h1>


     <!-- some other content here -->


</div>


in your css declaration you bind the different boxes with named page rules



.cover{


page :cover;


}


 

.toc{


page: toc;


}


 

.content{


page: content;


}


 

.blank{


page : blank;


 

}


then you declare style for each named page




@page cover{


   margin: 0;


   page-break-after: always;


 

}


 

@page blank{


 

/*content:normal will remove any dynamic content in the blank page like page number*/


  

 @top-left { content: normal }


 @bottom-left { content: normal }


 @top-right { content: normal }


 @bottom-right { content: normal }  

 page-break-after: always;


}


 

@page toc{


 page-break-after: always;


}


 

 

 

@page content{


 

/*we use -fs-page-sequence to tell the renderer starting or stoping counter*/


-fs-page-sequence: start;


 

/*then we tell to the renderer to insert page number on the bottom right of content page*/


 @bottom-right{


          content: "Page " counter(page) " of " counter(pages);


 }


 

}



Take a look at http://kent.w3.org/TR/css3-page for more page rule documentation
