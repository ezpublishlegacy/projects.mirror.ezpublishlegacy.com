OWMobile

Last updated: Wednesday 24 October 2012 15:51

UNIX name Owner Status Version Compatible with
owmobile Simon Alberny stable v2.0 4.x.x
OW Mobile is an ezpublish extension providing tools for mobile detection and user redirection.
Mobile detection and redirection can be processed either by PHP or JavaScript.
User can choose to visit the standard website even on a mobile device.
Both methods use detection regexp from http://detectmobilebrowsers.com/

How to use OWMobile ?

PHP method

  • Put this line on top of your pagelayout.tpl: {first_set( $module_result.view_parameters['site_type'], '' )|owmobileredirectionstarter()}
  • To set a link to the standard website with auto-redirect disabled, use viewparameter 'site_type' with value 'normal|iphone|ipad|mobile'
    eg. http://www.standard_website.com/(site_type)/normal
  • Be careful with reverse-proxys, JS method is safer in this case

JS method

thanks to https://github.com/sebarmeli/JS-Redirection-Mobile-Site

  • insert "design/standard/javascript/redirection_mobile.js" in your page head
  • include "design/standard/templates/redirection_mobile.tpl"
  • The JS method can receive a GET parameter to disable auto-redirect. The name of this parameter can be set in 'noredirection_param' ini parameter, in owmobile.ini (default value if not set is 'noredirection'). 'noredirection_param' GET parameter value must be string 'true'.
    eg. http://www.standard_website.com/?nomobredirect=true

Download

Please visit https://github.com/Open-Wide/OWMobile/tags to download latest version of owmobile.
