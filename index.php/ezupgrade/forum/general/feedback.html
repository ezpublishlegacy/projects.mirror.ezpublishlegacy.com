Also an option for 'yes' to all questions. 


This is now implemented. Well, It doesn`t say yes to all. But uses the default value.
The way to activate it is in account.ini:




[Accounts]


# Determine what action to make on prompts. Default is prompt, so you could answear on #each of them. Its recommended.


# Options are 

# prompt: ask what to do


# use_default: uses the default option


 

DefaultPromptOption=use_default
