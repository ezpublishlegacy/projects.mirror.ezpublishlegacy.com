Release 0.0.1 available

Brookins Consulting

Monday 29 October 2007 01:53

Brookins Consulting is pleased to announce the release of the Unofficial Hab.la extension a Hab.la based livehelp, instant message, chat solution for eZ Publish web sites.

The Unofficial Hab.la exension provides a clean simple solution to the common need of how to offer live help features with eZ Publish. Download0.0.1 today!

What is Hab.la?

Hab.la is a lightweight method to chat with visitors to your website using your existing instant messaging client. Visitors to your website appear as buddies on your Buddy list, their messages to you appear as IMs. Learn more about Hab.la take the tour, signup and get started.

Features

This is the first release of this solution which offers the following features

  • Hab.la: All of the hab.la features
  • eZ Publish: Enabled per siteaccess (on all pages) by default upon activation

Composition

A javascript design extension provides unofficial support of the hab.la website chat instant message software.
