SQLi Import

Last updated: Tuesday 05 July 2011 14:15

UNIX name Owner Status Version Compatible with
sqliimport Jérôme Vieilledent stable 1.2.1 4.1+
Yet another import extension :)

SQLIImport is an extension allowing to import external content in eZ Publish.

It provides a framework for content manipulation and a simple interface for developing import handlers, so as a GUI to administrate your imports in the admin interface

Import handlers are classes you need to develop in order to process external data (XML, CSV, ...) and import it into eZ Publish.

Sources are now available on GitHub : https://github.com/lolautruche/sqliimport
