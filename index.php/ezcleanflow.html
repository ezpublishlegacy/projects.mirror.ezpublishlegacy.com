eZ Flow - Developer Design

This extension pertains to both the eZ publish ezwebin as well as ezflow extensions. The extension aims to provide developers with a stripped down clean design for rapid design development.

As an integration company for eZ Publish we do many sites very often and found that it would be extremely useful to have a clean design that can be used with new clients instead of having to manually start overriding each file and removing the black round boxes etc.

This extension should not be used at any point in time for any demo application what so ever. The primary purpose is to only provide developers a faster more efficient way to install this design extension change the folder names to their respective project and start including the site's design that may have already been completed by a design team or another design company.
