ezworkflowcollection Gaetano Giunta beta 0.5 4.3+
A collection of general-purpose workflow events

A collection of general-purpose workflow events that can be used in many different setups, such as:

- multiplexer based on node location

- add secondary location to object

- add object to an object-relation

- send cache purge commands to reverse proxies such as squid or varnish

- approval of adding locations to an object

- object state update interface (not really a workflow event, but still useful)

- new dashboard blocks to display content waiting for publication (based on object state)

etc...
