HO Export

Last updated: Friday 14 September 2012 10:02

UNIX name Owner Status Version Compatible with
ho_export Harry Oosterveen stable 1.0 4.x
HO Export is an extension for eZ Publish to export (part of) an eZ Publish site for offline use, in particular to distribute on a CDROM.

HO Export exports all nodes in a sub-tree, branching out to main nodes if additional nodes are in the subtree. Th export includes binary files and images.

Typical use would be for a large site, adding a selection of existing object (using 'Add location') under a new node, and export these so they can be distributed on offline media, e.g. CDROM or USB stick.

The way it works is described in a blog post, see Create a CD-ROM from your eZ Publish site.
