silver.post

Last updated: Tuesday 20 March 2012 09:43

UNIX name Owner Status Version Compatible with
silver-post Frank Dege alpha 2012-03-09 4.5+
silver.post is an eZ extension that uses the "ezprestapiprovider" extension and adds the functionality to publish new articles to eZ Publish via the REST interface.

In combination with the companion Android and iPhone app silver.post it will be possible to submit articles on the go, for example blogging during a conference. The contribution contains a headline, short text and body text. An image can be added as well. The article can be categorized in predefined sections.

More information about the silver.post extension and the corresponding apps can be found at www.silversolutions.de/silver_post.
