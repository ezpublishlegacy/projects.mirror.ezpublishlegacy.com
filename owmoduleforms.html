OW Module Forms

Last updated: Friday 28 January 2011 22:20

UNIX name Owner Status Version Compatible with
owmoduleforms Franck Magnan beta 1.0 4.4
This extension is a form builder for your modules.

It provides a set of skeleton classes to extend to make your forms in a module/view way
It includes:

  • form structure definition: you do not need to write html, only configure input fields in your form class
  • extended validation: a set of predefined validators is provided with possibility to extend them to make your own validators
  • ajax validation with jquery library
  • form token use to prevent from CSRF exploit
