BC GMaps Location

Last updated: Wednesday 30 November 2011 03:58

UNIX name Owner Status Version Compatible with
bcgmapslocation Brookins Consulting stable 1.0.0 4.0+, 4.1+
A simple design/datatype extension which displays eZ Publish content objects on Google Map

Features

  • Updated: Templates which display child items are displayed on a custom Google Map with extended information (by adding complex content object data to new points on the map, each with a neat pop-up information box).
  • Updated: Google Maps Location Datatype

Distribution

  • https://github.com/brookinsconsulting/bcgmapslocation
