Orange Delight, a style package for the eZ Publish Website Interface.

Brookins Consulting is very proud to announce our own eZ Publish site style,
'Orange Delight' as an entry in the eZ design contest 2007.
Orange Delight is a complete eZ Publish site design package based on
the eZ Publish Website Interface.
You can test, examine and review the design using this eZ publish 3.9
demo with the Orange Delight site package installed and enabled.
eZ Publish customers and users are free to use this design for their
own websites.
Brookins Consulting, Silver Partner from North America / United States.
Brookins Consulting is certified by eZ system's to provide advanced
eZ Publish and eZ Publish Now development and design creating the
finest eZ Publish advanced solutions.
